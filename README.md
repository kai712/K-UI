> A Vue.js 2.0 UI Toolkit for Web

## Install
```shell
npm install ui-kk -S
```

or

```shell
yarn add ui-kk
```

## Quick Start
``` javascript
import Vue from 'vue'
import KUI from 'ui-kk'

Vue.use(...KUI)

// or
import {
  KButton
  // ...
} from 'ui-kk'

Vue.component(KButton)
```
